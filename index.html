What does this website do? Show me a few example prompts.

  • This website provides AI-generated insights using logs and metrics from this exact application.
  • It analyses data produced by the platform's own microservices, including the Log Insight Service, frontend gateway, and supporting workloads.
  • Logs come from the EFK stack, and metrics come from Prometheus running inside the same Kubernetes environment.
  • The Log Insight Service fetches recent data, prepares context, and sends it to an AWS Bedrock model for summarisation.
  • Limits for demo use : up to 5 queries per device per day | short prompts only | analysis restricted to the last 12 hours of logs and metrics
  • Example prompts : "Show key errors in the last 2 hrs" | "Summarise recent issues for this app"
  • The system integrates with multiple data sources to provide comprehensive analysis and insights.
  • Real-time monitoring capabilities allow for immediate detection and alerting of critical issues.
  • Advanced filtering options enable users to narrow down results based on specific criteria and timeframes.
  • Historical data retention ensures that long-term trends and patterns can be analyzed and tracked.
  • Custom dashboards can be created to visualize key metrics and performance indicators relevant to your application.
  • Integration with external tools and services extends the platform's capabilities and workflow automation.
